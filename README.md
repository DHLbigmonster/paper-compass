# Paper Compass

`Paper Compass` 是一个面向实证类学术论文写作的 Codex skill。

它的目标不是替用户代写整篇论文，而是把论文流程拆成可执行的阶段，引导用户完成文献阅读、实验设计、数据处理、结果分析、正式写作、LaTeX 排版与 AI 痕迹优化。

## 适用场景

- 实证论文
- 毕业论文
- 量化研究
- 需要数据实验的学术写作

## 仓库结构

- `SKILL.md`：主 skill 定义与引导流程
- `references/`：按阶段拆分的详细参考文档
- `agents/openai.yaml`：界面元数据

## 使用方式

把这个目录作为 skill 安装后，直接通过 `$paper-compass` 调用即可。
